# Python_GUI_app_for_expense_tracking_and_visualization
💰 Expense Visualiser (Python GUI)

A desktop-based expense tracking and visualization application built using Python, CustomTkinter, and Matplotlib.
The app allows users to input monthly expenses, enforces basic budgeting rules, and visualizes spending using a pie chart.

🚀 Features

Modern GUI using CustomTkinter

Background image support

Input validation for numeric values

Rule-based budget warnings:

⚠️ Food expense > 10% of net salary

⚠️ Rent expense > 20% of net salary

⚠️ Savings < 20% of net salary

Interactive pie chart visualization

Clear input fields with one click

Lightweight and beginner-friendly

🖥️ Tech Stack

Python 3

CustomTkinter

Matplotlib

Pillow (PIL)

Tkinter (messagebox)

📊 How It Works

Enter Net Salary

Add expense name (e.g., food, rent, savings)

Enter expense amount

The app:

Validates input

Shows warnings if budget rules are violated

Stores expense data

Click Plot to generate a pie chart showing expense distribution
